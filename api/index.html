  • SUB hangboard/workout/setinfo

    Get information on the current set.

    Operation IDGetSetInfo

    Accepts the following message:

    Current Set
    object
    resttostart
    integer
    >= 0

    Rest to start time in seconds.

    exercise
    string

    Exercise short description.

      Examples values:
    • "Hang"
    • "Pull Up"
    counter
    integer
    >= 0

    How long to run in seconds or how much repetitions (for pull ups).

    pause
    integer
    >= 0

    Pause between the exercises in seconds.

    reps
    integer
    >= 0

    Number of exercise repetitions in this set.

    left
    string

    Left hold to use.

      Examples values:
    • "A1"
    • "JUG"
    right
    string

    Right hold to use.

      Examples values:
    • "A7"
    • "EDGE"
    type
    string

    Type of exercise.

      Examples values:
    • "Hang"
    • "Maximum Hang"
    • "Pull Up"
    intensity
    number

    Hang intensity if applicable for this exercise.

    Additional properties are allowed.

    Examples

    {
  "resttostart": 5,
  "exercise": "Hang",
  "counter": 10,
  "pause": 5,
  "reps": 3,
  "left": "A1",
  "right": "A7",
  "type": "Hang",
  "intensity": 0
}
    This example has been generated automatically.

  • SUB hangboard/workout/timerstatus

    Information on the current timer status.

    Operation IDGetCurrentTimerStatus

    Accepts the following message:

    Timer Status
    object
    Duration
    number

    Timer duration in seconds.

    Elapsed
    number

    Timer elapsed time in seconds.

    Completed
    number

    Relative completion of the current timer - elapsed / duration.

    Countdown
    number

    Countdown number - count every second downwards. Will be -1, when no full second crossed.

      Examples values:
    • -1
    • 10
    • 9
    • 8
    CurrentSet
    integer

    Current set index in this workout.

    TotalSets
    integer

    Total sets in workout.

    CurrentRep
    integer

    Current repetition index in this set.

    TotalReps
    integer

    Total repetitions in this set.

    Additional properties are allowed.

    Examples

    {
  "Duration": 20,
  "Elapsed": 3,
  "Completed": 0.8,
  "Countdown": -1,
  "CurrentSet": 1,
  "TotalSets": 5,
  "CurrentRep": 2,
  "TotalReps": 5
}
    This example has been generated automatically.

  • SUB hangboard/sensor/load/loadstatus

    Information on the current load status including the calculated values.

    Operation IDGetCurrentLoadStatus

    Accepts the following message:

    Load Status
    object
    time
    integer

    Timestamp in seconds.

    loadcurrent
    number

    Current load in kg.

    loadaverage
    number

    Average load in kg.

    loadmaximal
    number

    Maximal load in kg.

    loadloss
    number

    Relative loss of load - current / maximal.

    fti
    number

    Force Time integral

    rfd
    number

    Rate of force development

    Additional properties are allowed.

    Examples

    {
  "time": 123412341234,
  "loadcurrent": 23.2,
  "loadaverage": 55.3,
  "loadmaximal": 99.2,
  "loadloss": 0.2,
  "fti": 231.1,
  "rfd": 12.2
}
    This example has been generated automatically.
