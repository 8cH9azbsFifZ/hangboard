# Exercises
This directory contains the exercise files and the corresponding classes for serving the timers.

## Preparation
```
python3 -m venv venv
source venv/bin/activate
python3 -m pip install -r requirements.txt
```

# Running the backend
```
./startup.sh
```

wscat -c "ws://127.0.0.1:4321/"
