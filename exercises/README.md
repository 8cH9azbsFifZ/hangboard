# Exercises
This directory contains the exercise files and the corresponding classes for serving the timers.

# Running the backend
```
python3 -m venv venv
source venv/bin/activate
python3 -m pip install -r requirements.txt
HOST=127.0.0.1
PORT=9090
PORTRECV=9091
export HOST
export PORT
export PORTRECV
python3 exercise.py
```
