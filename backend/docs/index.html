  • SUB hangboard/workout/setinfo

    Get information on the current set.

    Operation IDGetSetInfo

    Accepts the following message:

    Current Set
    object
    resttostart
    integer
    >= 0

    Rest to start time in seconds.

    exercise
    string

    Exercise short description.

      Examples values:
    • "Hang"
    • "Pull Up"
    counter
    integer
    >= 0

    How long to run in seconds or how much repetitions (for pull ups).

    pause
    integer
    >= 0

    Pause between the exercises in seconds.

    reps
    integer
    >= 0

    Number of exercise repetitions in this set.

    left
    string

    Left hold to use.

      Examples values:
    • "A1"
    • "JUG"
    right
    string

    Right hold to use.

      Examples values:
    • "A7"
    • "EDGE"
    type
    string

    Type of exercise.

      Examples values:
    • "Hang"
    • "Maximum Hang"
    • "Pull Up"

    Additional properties are allowed.

    Examples

    {
  "resttostart": 5,
  "exercise": "Hang",
  "counter": 10,
  "pause": 5,
  "reps": 3,
  "left": "A1",
  "right": "A7",
  "type": "Hang"
}
