# Hardware setup

HC-SR04 - Raspi
VCC - Pin 2 (VCC)
GND - Pin 6 (GND)
TRIG - Pin 12 (GPIO18)
ECHO - R1: 330Ω 
R1: 330Ω  - Pin 18 (GPIO24) 
R1: 330Ω -  R2: 10kΩ 
R2: 10kΩ -  Pin6 (GND)
