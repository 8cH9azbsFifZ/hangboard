
# Running
```
yarn install
yarn start
yarn run ios
yarn run android
yarn run web
```

# Development information
## Initial setup
npx react-native init hangboardapp

## Build
yarn install
cd ios && pod install && cd ..

## Add sounds
iOS: Open Xcode and add your sound files to the project (Right-click the project and select Add Files to [PROJECTNAME])
